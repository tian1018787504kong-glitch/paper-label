# Changelog

## 0.1.0

Initial open-source baseline for paper-label.

- Browser extension for local journal labels.
- Local ranking dataset import.
- Multiple ranking badges per paper.
- Badge color, size, and display-order settings.
- Local literature library with folders, tags, notes, and export.
- Lawful full-text search and DOI navigation.
- Interface language setting with follow-system, Chinese, and English modes.
- Chromium, Firefox, and Safari-oriented build scripts and upload artifacts.
- No bundled ranking dataset, account system, cloud sync, translation, Zotero sync, membership, payment, or paywall-bypass download feature.
