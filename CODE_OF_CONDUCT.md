# Code of Conduct

paper-label aims to be a practical, respectful open-source project for researchers, students, and developers.

Please:

- Be kind and specific in feedback.
- Assume good intent, but help correct mistakes clearly.
- Keep discussions focused on reproducible issues and useful improvements.
- Do not post private data, copyrighted datasets without permission, credentials, or paywall-bypass instructions.

Maintainers may close or remove comments that are abusive, discriminatory, spammy, or legally unsafe.
