# Roadmap

## Open-source foundation

- Improve metadata extraction for major publisher article pages.
- Improve search-result page precision so labels only show when the source journal is confidently recognized.
- Add more automated adapter tests.
- Improve local library import/export compatibility.
- Improve accessibility and keyboard operation.

## Not in this open-source package

These features are intentionally outside this repository:

- Shared ranking dataset service.
- User accounts and cloud sync.
- Paid translation.
- Zotero sync.
- Membership and payment.

If those are built later, they should live in separate private repositories or clearly separated packages.
