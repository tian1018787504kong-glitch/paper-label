# paper-label 用户手册 / User guide

English version: [USER_GUIDE.en.md](USER_GUIDE.en.md).

这份手册面向第一次使用 paper-label 的用户。英文界面使用同一套操作流程，页面文字会根据语言设置显示。

## 1. 选择界面语言

打开扩展设置，在“界面语言 / Interface language”中选择：

- 跟随系统 / Follow system：优先使用浏览器 UI 语言；
- 中文 / Chinese；
- English。

选择后刷新设置页或重新打开扩展即可生效。跟随系统模式下，浏览器语言优先于网页语言。

## 2. 导入评级数据

1. 打开“期刊标签数据管理”。
2. 点击“选择文件”，选择 JSON 文件。
3. 等待页面显示“待导入文件”和数据集数量。
4. 检查数据集名称、版本和记录数。
5. 点击“确认导入”。
6. 在数据集卡片中打开“启用”和“页面显示”。
7. 如需马上更新已经收藏的论文，点击“批量重算期刊标签”。

数据集不会上传服务器。数据格式见 [DATASET_FORMAT.md](DATASET_FORMAT.md)。

## 3. 在论文网页上使用

在支持的学术搜索页或论文详情页：

- 标签会出现在论文标题附近；
- 右侧悬浮球提供“收藏文献”和“查找全文”；
- 悬浮球可以拖到不遮挡正文的位置；
- 论文匹配多个评级体系时，会显示多个标签；
- 没有匹配到已启用数据集时，不显示空标签框。

网站结构差异很大。如果某个页面没有显示，请先确认：扩展已打开、站点访问权限为“所有网站”、对应数据集已启用且期刊名称/ISSN 在数据集中存在。仍有问题时，请提交页面 URL、浏览器版本和可公开的复现步骤，不要上传账号、Cookie 或受限全文。

## 4. 管理本地文献库

点击扩展图标并打开“本地文献库”：

- 列表页用于搜索、筛选和批量选择；
- 详情面板用于修改标题、作者、年份、期刊、DOI、URL、分类、标签和备注；
- 一篇文献可以属于多个分类；
- 标签在输入空格或回车后生成；
- “取消收藏”会从本地文献库移除该条记录；
- “打开原文页面”只打开原论文 URL，不下载 PDF。

## 5. 导出与备份

在文献库页面选择 JSON、CSV、BibTeX 或 RIS 导出。建议定期导出 JSON 作为完整备份；CSV、BibTeX 和 RIS 更适合交给其他文献工具继续处理。

## 6. 查找全文

“查找全文”只打开合法来源或入口，例如 DOI 出版社页、开放获取检索、PubMed Central 或学校图书馆入口。需要机构权限的论文，请在出版社或学校页面自行登录。
